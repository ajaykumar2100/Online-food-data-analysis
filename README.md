# Online-food-data-analysis
Analyzed online food ordering dataset to identify trends and customer behaviour.

Project Overview

This project analyzes an online food ordering dataset to uncover customer ordering behavior, restaurant performance, delivery outcomes, and payment preferences. The goal was to generate actionable insights that can help improve customer experience and business operations.

Objectives

- Analyze order status trends (Delivered, Cancelled, Delayed)
- Compare restaurant ratings and customer satisfaction
- Identify popular food categories
- Study payment method preferences
- Evaluate restaurant-wise order performance
- Understand city-wise order distribution

Tools Used

- Python (data cleaning and preprocessing)
- Pandas
- NumPy
- Microsoft Power BI (dashboard creation)

Dashboard Insights

Order Status Analysis

- Compared average orders across Delivered, Cancelled, and Delayed statuses.
- Helped identify delivery performance trends.

City-wise Order Distribution

- Analyzed order count across multiple cities.
- Useful for identifying top-performing locations.

Food Category Trends

- Italian and Chinese cuisines showed higher average orders.
- Indian cuisine had relatively lower average orders.

Payment Method Analysis

- Wallet payments had the highest average usage.
- UPI, Card, and Cash were also widely used.

Restaurant Performance

- Compared average orders by restaurant.
- Helped identify best-performing outlets.

Customer Ratings

- Analyzed average customer ratings across restaurants.
- Most ratings were in the mid-to-high satisfaction range.

Key Outcomes

- Identified customer food preferences and payment habits.
- Highlighted restaurant performance differences.
- Provided insights into delivery and operational efficiency.

Files Included

- Food Orders Analysis Dashboard (.pbix)
- Cleaned dataset (.csv)
- Dashboard screenshots
- Project documentation

Conclusion

This project demonstrates how data analysis and visualization can be used to improve food delivery operations, customer satisfaction, and business decision-making.
